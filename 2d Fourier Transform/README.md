### 二位离散傅里叶变换（图像傅里叶变换）

    定义：二维离散傅里叶变换是将图像从空域（space domain）转换至频域（frequency domain）的变换。
    应用：图像增强、图像去噪、图像边缘检测、图像特诊提取、图像压缩等。
    
    理论依据：传统的傅里叶变换
